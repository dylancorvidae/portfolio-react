PROJECTS

Basic Ass Maze Race

Description:

A SPA multiplayer game built using Socket.io, React, and p5 that allows players to invite up to eight friends to a private lobby and race against each other to see who can solve a randomly generated maze the fastest. Built by a team of 4 in 6 days.

My Role:

As UI & UX design lead I shaped how our game looked and was experienced by the end-users. I built accessibility into this game for people who live with different types of color blindness and it meets the WCAG standards for contrast and color. I utilized Coblis, a color blindness simulator, to test my color palettes and screenshots of the game throughout the production. In addition to UI & UX I wrote all of the CSS, a large portion of the HTML, built some of the React components, and wrote some of the tests. I wrote and organized the supporting docs that our group worked off as we built our app.
